# GestorNotas

Este es un proyecto en Java para gestionar notas de usuarios con registro y login.  

## Características

- Registro de usuarios con contraseña (guardada en hash SHA-256)  
- Login de usuario  
- Crear, editar y eliminar notas  
- Ver lista de notas  
- Persistencia de notas por usuario en archivos `.txt`  
- Guardado automático de notas  
- Cerrar sesión y volver al login  

## Estructura de archivos
